# Nublus - Prototipo de Gestión de Flotas

Este proyecto es un prototipo interactivo para administrar vehículos, controlar el vencimiento de documentos y visualizar alertas. Fue desarrollado originalmente para la materia de **Gestión de Requisitos de Sistemas** en el programa de **Técnico en Servicios de Seguridad Informática**.

## ¿Qué hace el proyecto?
* **Administración de datos**: Permite agregar y buscar vehículos por placa, marca y modelo.
* **Control de vencimientos**: Clasifica el estado de los documentos de cada vehículo en vigentes, próximos a vencer y vencidos.
* **Persistencia local**: Guarda y carga la información utilizando `localStorage`, para que los datos no se borren al recargar la página.
* **Interfaz de usuario**: Utiliza HTML y CSS estructurado para dar soporte a las funciones operativas del prototipo.

## Archivos del repositorio
* `index.html`: Estructura base de la aplicación.
* `styles.css`: Hojas de estilo adaptadas para un tema oscuro.
* `app.js`: Lógica en JavaScript que maneja el almacenamiento y las interacciones.
* `docs/`: Carpeta con el documento de análisis que detalla los requisitos del prototipo.
